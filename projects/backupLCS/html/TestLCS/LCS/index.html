A family of elements of R (indexed by I) that satisfy the following properties:

    The sum of all the elements is 1 (completeness)

    Each element is idempotent (applying it twice is the same as applying it once)

    Distinct elements are orthogonal (applying one after the other gives 0)

    Each element is self-adjoint (Hermitian condition)

structure IsMeasurementSystem {I : Type*} [Fintype I] (f : I → R) : Prop where sum_one : ∑ x, f x = 1 idempotent : ∀ x, f x * f x = f x orthogonal : ∀ x y, x ≠ y → f x * f y = 0 self_adjoint : ∀ x, star (f x) = f x

A strategy for an LCS game consists of:

    For each question i, and each possible assignment of values to the variables in V i, we have an element of R (this is the E function).

    For each variable j, and each possible outcome (0 or 1), we have an element of R (this is the F function).

    For each question i, the family of elements given by E i forms a measurement system.

    For each variable j, the family of elements given by F j forms a measurement system.

    For each i, j, and each possible assignment α and β, E_(i,α) commutes with F_(j,β).
